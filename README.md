# Practice Projects

## 📁 プロジェクト一覧

ライトのON/OFFを切り替えるシンプルなJavaScriptアプリ。

### ✊✋✌️ じゃんけんアプリ

→ `02.rps-game/`

### ✈️ 旅行ジェネレーター
